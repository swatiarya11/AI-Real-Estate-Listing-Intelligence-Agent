# AI Real Estate Listing Intelligence Agent

## Project Overview

AI-powered real estate analysis platform built using:

- Streamlit
- n8n
- Groq LLM
- RapidAPI
- Google Sheets
- Telegram

## Features

- Property Listing Dashboard
- Investment Score Calculation
- AI Property Analysis
- High Potential Property Detection
- Google Sheets Integration
- Telegram Alerts

## Workflow

Webhook
→ Property API
→ Data Cleaning
→ Property Mapping
→ Investment Score Engine
→ AI Analysis
→ Telegram Alert
→ Google Sheets

## Technology Stack

- Python
- Streamlit
- n8n
- Groq
- RapidAPI
- Google Sheets
